# Editor Mappe Concettuali — costruzione libera

Apri `index.html` in un browser moderno (Chrome/Edge/Safari/Firefox).

## Cosa puoi fare
- Aggiungere nodi di testo dove vuoi
- Aggiungere nodi immagine (caricamento locale) e scrivere una didascalia
- Trascinare nodi liberamente
- Collegare nodi con frecce + etichetta (strumento "Collega")
- Pan nello spazio “infinito” (strumento "Pan" o barra spazio premuta)
- Zoom con rotellina del mouse
- Inspector a destra per modificare: testo, colori, larghezza, etichette frecce
- Salva/Carica nel browser (LocalStorage)
- Import/Export JSON

## Note
- Offline: nessun server.
- Le immagini vengono salvate nel progetto come dataURL dentro il JSON / LocalStorage (comodo, ma può diventare pesante con tante immagini).
