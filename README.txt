# Mappe Concettuali — Editor Touch (iPad)

Apri `index.html` su iPad (Safari/Chrome).

## Come si usa (pensato per touch)
- **Sposta**: trascina i nodi con un dito.
- **Due dita**: pan in qualunque momento.
- **Pinch**: zoom in/out.
- **Pan**: abilita il pan con un dito sullo sfondo (utile se vuoi pan “preciso”).
- **Collega**: tocca un nodo, poi un altro. (Seleziona la freccia per modificarne l’etichetta.)
- **+ Testo**: poi *tocca lo sfondo* dove vuoi inserirlo.
- **+ Immagine**: scegli immagine, poi *tocca lo sfondo* dove vuoi inserirla.
- **Tieni premuto** su un nodo o sull’etichetta di una freccia per modificarli (iPad-friendly).
- **Elimina**: elimina nodo o freccia selezionata.

## Salvataggio
- Salva/Carica nel browser (LocalStorage)
- Import/Export JSON

## Nota
Le immagini vengono memorizzate come dataURL nel progetto: comodo, ma se metti molte immagini il file JSON può crescere.
