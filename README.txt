# Costruttore di Mappe Concettuali (web app)

Apri `index.html` in un browser moderno.

## Funzioni
- Wizard passo-passo (concetto principale → rami → sotto-concetti)
- Anteprima SVG in tempo reale
- Trascinamento nodi (on/off)
- Doppio click su nodo per rinominare
- Click su freccia o etichetta per cambiare relazione
- Salva/Carica nel browser (LocalStorage)
- Import/Export JSON
- Export SVG e PNG

## Note
La web app è completamente offline: nessun server, nessuna dipendenza.
